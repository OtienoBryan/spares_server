# Required Package Installation

To complete the staff authentication setup, you need to install the following packages:

## Install Required Packages

```bash
cd backend
npm install bcrypt @nestjs/jwt @types/bcrypt
```

## Package Details

- **bcrypt**: For password hashing
- **@nestjs/jwt**: NestJS JWT module for token generation and validation
- **@types/bcrypt**: TypeScript types for bcrypt

After installation, restart your backend server.
